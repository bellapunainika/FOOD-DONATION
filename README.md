# 🍱 FOOD DONATION

A full-stack web app connecting food donors, NGOs, and delivery 
volunteers to reduce food wastage.

## Features
- Real-time interactive map with urgency-based prioritization
- Role-based authentication (Donor, NGO, Volunteer)
- AI chatbot powered by Google Vertex AI Gemini
- Serverless architecture on Firebase

## Tech Stack
React, TypeScript, Tailwind CSS, Firebase, Vertex AI, React-Leaflet
